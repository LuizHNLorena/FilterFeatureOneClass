Introduction

Filters developed for the article "Filter Feature Selection for One-Class Classification" avaiable at Journal of Intelligent & Robotic Systems

The following filters are avaiable:

  • Information Score
  • Pearson Correlation
  • Intra-class Distance
  • Interquartile Range

Spectral score is avaiable at Arizona State University Feature Selection Algorithms

Usage

File howToUse.m present how to use the filters in the data set iris1.data. This data set represents class "Setosa" from the Iris Data Set
